# テプラで作るウォッチバンドカレンダー

このリポジトリは、テプラでプリントしてウォッチバンドに貼れるカレンダー画像、およびそれを生成する Python プログラムを提供します。

## 概要

- `calendars/` ディレクトリにテプラ用の画像を用意しています
  - 生成画像は縦 96 ドットで、18mm テプラでのプリントに最適です
  - 縦 96 ドットの要件を満たしていれば、テプラ以外のラベルライターでも使用できます
- 日本の祝日に対応しています

## 使用方法

1. `calendars/` ディレクトリから、使用したい年月のカレンダーをダウンロードしてください。
2. テプラのアプリで画像を読み込み、プリントをしてください。
3. プリントしたラベルを腕時計のバンドなどに貼り付けてご使用ください。

## ファイル構成

- `tepra_calendar.py` - カレンダー画像を生成するメインスクリプト
- `requirements.txt` - 依存パッケージ
- `calendars/` - 生成した画像の出力先
- `fonts/` - 使用するフォントファイル
- `LICENSE` - MIT ライセンス

## 実行方法

### 要件

- Python 3
- Pillow
- jpholiday

`requirements.txt` に必要なパッケージを記載しています。

### 実行

1. リポジトリをクローンします
2. 依存関係をインストールします

```bash
pip install -r requirements.txt
```

3. プログラムを実行します

```bash
python tepra_calendar.py 2025 2025
```

この例では、`2025` 年の 1 月から 12 月までのカレンダー画像が `calendars/` フォルダに生成されます。
